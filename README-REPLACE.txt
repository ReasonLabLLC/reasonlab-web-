REASONLAB WEBSITE STUDIO — VERSION 2

This package replaces the first Website Studio files.

REPLACE:
- websites.html
- css/websites.css
- js/websites.js
- the complete demos folder

IMPORTANT:
The old shared file demos/demo.css is no longer used.
You can delete it after replacing the demos folder.

Each demo now has its own:
- index.html
- style.css
- app.js

This creates five genuinely different experiences:
1. Savoria Kitchen — editorial restaurant + menu tabs + reservation flow
2. PrimeFix Home Services — bold local service + quote wizard + areas/reviews
3. Sterling Advisory Group — premium professional + case studies + consultation
4. Élan Beauty Studio — fashion-led beauty + service filters + artist booking
5. Nova Dental Care — patient-first dental + treatments + insurance + appointment flow

The Website Studio page is now light/editorial to distinguish it from the dark ReasonLab landing page.
All five showcase cards have equal size and identical Explore Demo buttons.
The analytics and heatmap section remains a dark ReasonLab-style feature block.

The demo websites load only after the visitor clicks Explore Demo.
